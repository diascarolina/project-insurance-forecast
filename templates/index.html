Health Insurance Price Forecast

Welcome to the Health Insurance Price Forecast. In this project, we used linear regression to predict how much would someone pay in insurance using some variables related to personal information, such as age and gender.

You can make a POST request to the URL https://insurance-forecast.herokuapp.com/predict/ using the following json as an example:

{ "age": 19, "sex": "female", "bmi": 25, "children": 1, "smoker": "no", "region": "northwest" }
